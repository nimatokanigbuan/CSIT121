My Introduction to HTML

Lab 1 & 2

This week was my first time exploring HTML. Although it's pretty tedious it can be entertaing as well. If I were to give advice to someone just starting, I would recommend practicing every time they learn about a new element. By doing this, I found that some things started to feel a habit, and it's only been a week!

Things I've Learned

  1. HTML is a coding language that defines the structure and meaning of content on a web page.
  2. Web Standards exist to ensure websites work on all devices and enhance the online experience for everyone.

Things I've Learned Part 2

  1. External stylesheets allow you to change the appearance of multiple HTML pages by linking them to a single CSS file.
  2. CSS can be used to control text styling, such as font types, colors, and layout, improving the user experience and design.
