Why is it Important?

A key benefit of Micropub is that it encourages interoperability between different tools and platforms.

The Micropub protocol is a part of the IndieWeb movement. This movements goal is to give users more control over their online content and data by using user-owned web services.

Using Micropub gives users the freedom to choose or build their preferred writing tools without being locked into a specific blogging system.
